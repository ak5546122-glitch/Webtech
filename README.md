# Webtech

Web Technology Practical Assignment
Description

This project is made for the Web Technology practical assignment. It helps to understand the basic concepts of web development. The project uses HTML for creating the webpage structure, CSS for designing and styling, and JavaScript for adding simple functionality.

This assignment shows how a basic website works and how different web technologies are used together.

Technologies Used

HTML

CSS

JavaScript

Objective

To learn basic webpage design

To understand simple styling using CSS

To apply basic JavaScript concepts

How to Run

Open the index.html file in any web browser.
